## Navigation Guide

### Frontend Routes
- **Landing Page**: `/`
- **Login**: `/auth/login`
- **Signup**: `/auth/signup`
- **Products**: `/products`
- **Cart**: `/cart`

### key Features
- Authenticated users can add items to cart and checkout.
- Checkout creates an Order in the database.
